DevOps Technical radar

The Radar is a document that sets out the changes that are currently interesting regarding software development within UKHSA.
It is a knowledge base about where we are today as an agency and the tools we want to move towards going forward.

Aim of this tech radar

This technical radar is designed to help guide the UKHSA suppliers on technologies and frameworks the agency wants to embrace going forward.
The original concept comes from thoughtworks, you can read more about it here.

These solutions may bring values. It is necessary to assess and test them in our environment to confirm or not this vision.

For various reasons, these solutions are no longer the first choices or are not recommended (anymore).
